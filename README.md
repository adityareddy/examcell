examcell
========

Automation of Exam Cell

Installation instructions
=========================

1) Create an environment directory:
mkdir projenv
virtualenv projenv
source projenv/bin/activate

2) Fetch source code:
git clone https://github.com/ratanraj/examcell.git

3) Installing dependencies:
Go to examcell directory.
pip install -r requirements.txt

You are ready to go.
